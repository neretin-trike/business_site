## Business

Статический лендинг для тренировки вёрстки. 

Используемые технологии: 
+ Foundation 6;
+ Sass;
+ Grunt;

Необходимые инструменты для запуска в режиме разработки:
+ node.js
+ grunt

### Скачать все зависимости
`npm install`

### Запустить проект в режиме отслеживания изменения файлов
`grunt`
